6.7.3.4 Extended Runtime Systems

An extended runtime system is a variant of a runtime system with additional capabilities, including the presence of the Prolog compiler. Extended runtime systems are created with spld in a way similar to how ordinary runtime systems are created. An extended runtime system requires a license; see Managing Extended Runtime License Information in SICStus Prolog Release Notes for details about managing such license information.
