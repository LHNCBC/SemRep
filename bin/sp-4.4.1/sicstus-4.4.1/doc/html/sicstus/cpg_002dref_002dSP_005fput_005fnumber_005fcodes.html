12.3.80 SP_put_number_codes()

Synopsis

#include <sicstus/sicstus.h>

int
SP_put_number_codes(SP_term_ref term,
                    char const *s);

Assigns to term a reference to a Prolog number obtained by parsing s as if by number_codes/2.

Arguments

term

The SP_term_ref to be assigned

s

The string to parse

Return Value

Zero if the conversion fails (as far as failure can be detected), and a nonzero value otherwise.

See Also

Creating Prolog Terms.
